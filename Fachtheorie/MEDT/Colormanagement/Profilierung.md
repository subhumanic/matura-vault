# Profilierung von Ausgabegeräten

Systeme enthalten: Testdruck-Datei und Messgerät

Prozess:
- Testdatei (Soll-Werte) -> Testdruck 
- Testdruck -> Messgerät -> Ist-Werte
- Ist-Werte + Soll-Werte -> CMM -> **ICC-Profil**
